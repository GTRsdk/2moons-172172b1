<h2>The installer is disabled</h2>
<p>In order to enable the installer, ENABLE_INSTALL_TOOL must be present in the includes/ folder./p>
<ul>
    <li>In the includes/ folder, create ENABLE_INSTALL_TOOL. The file name must be uppercase and the file itself should not be empty.</li>
</ul>
<p>For security purposes, it is highly recommended to rename the file or delete it when you are finished with installation.</p>
